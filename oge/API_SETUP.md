# Подробная инструкция: Как получить API_ID и API_HASH

## Шаг 1: Откройте сайт
Перейдите на https://my.telegram.org/apps

## Шаг 2: Вход
1. Введите ваш номер телефона в формате: **+79991234567** (с плюсом и кодом страны)
2. Нажмите "Next" или "Далее"
3. В Telegram вам придет код подтверждения
4. Введите этот код на сайте

## Шаг 3: Создание приложения (если нужно)

После входа вы увидите страницу **"App configuration"**.

### Если вы видите пустую страницу или кнопку "Create new application":
1. Нажмите **"Create new application"** (Создать новое приложение)
2. Заполните форму:
   - **App title** (Название приложения): 
     * Должно быть на английском языке
     * Только буквы, цифры и пробелы
     * Примеры: `My Reposter`, `Telegram Bot`, `Channel Reposter`
     * ❌ НЕ используйте: русские буквы, спецсимволы (!@#$% и т.д.)
   
   - **Short name** (Короткое имя): 
     * Только английские буквы (a-z), цифры (0-9) и подчеркивание (_)
     * Должно начинаться с буквы
     * Примеры: `reposter`, `mybot`, `channel_bot`
     * ❌ НЕ используйте: русские буквы, дефисы (-), пробелы, спецсимволы
   
   - **URL** (необязательно): можно оставить пустым
   
   - **Platform**: выберите `Other` (Другое)
   
   - **Description** (необязательно): можно оставить пустым
   
3. Нажмите **"Create application"** (Создать приложение)

**Важно:** Если видите ошибку "Incorrect app name!":
- Убедитесь, что используете только английские буквы
- В Short name не должно быть пробелов, дефисов и спецсимволов
- Попробуйте простые варианты: `reposter`, `mybot`, `telegrambot`

### Если у вас уже есть приложение:
Вы сразу увидите данные на странице.

## Шаг 4: Копирование данных

После создания (или если приложение уже было), на странице вы увидите:

```
App configuration

App api_id: 12345678
App api_hash: abcdef1234567890abcdef1234567890
```

**Что нужно скопировать:**

1. **App api_id** - это **число** (например: `12345678` или `9876543`)
   - Скопируйте это число
   - В `config.py` вставьте БЕЗ кавычек: `API_ID = 12345678`

2. **App api_hash** - это **строка** из букв и цифр (например: `abcdef1234567890abcdef1234567890`)
   - Скопируйте всю строку
   - В `config.py` вставьте В КАВЫЧКАХ: `API_HASH = 'abcdef1234567890abcdef1234567890'`

## Шаг 5: Заполнение config.py

Откройте файл `config.py` и замените:

```python
# Было:
API_ID = 12345678  # Замените на ваш API ID (число, без кавычек!)
API_HASH = 'YOUR_API_HASH'  # Замените на ваш API Hash (строка в кавычках)

# Стало (пример):
API_ID = 12345678  # Ваш реальный API ID
API_HASH = 'abcdef1234567890abcdef1234567890'  # Ваш реальный API Hash
```

## Пример правильного заполнения:

```python
API_ID = 12345678
API_HASH = 'abcdef1234567890abcdef1234567890'
SOURCE_CHANNEL = '@source_channel'
TARGET_CHANNEL = '@your_channel'
YOUR_CHANNEL_LINK = 'https://t.me/your_channel'
CHECK_INTERVAL = 10
```

## Важно!

- ❌ **НЕ** публикуйте ваш `api_hash` в открытом доступе
- ❌ **НЕ** делитесь файлом `config.py` с другими
- ✅ Храните `config.py` в безопасности
- ✅ Файл `config.py` уже добавлен в `.gitignore` для защиты

## Если что-то не получается:

1. **Не вижу кнопку "Create new application"**:
   - Возможно, у вас уже есть приложение - прокрутите страницу вниз, данные должны быть там

2. **Не могу войти**:
   - Убедитесь, что используете правильный номер телефона
   - Проверьте, что код подтверждения введен правильно
   - Попробуйте через другой браузер

3. **Не вижу api_id и api_hash**:
   - Убедитесь, что вы создали приложение
   - Попробуйте обновить страницу (F5)
   - Проверьте, что вы вошли в правильный аккаунт
