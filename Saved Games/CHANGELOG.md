# Changelog

## Версия 1.0.0 (2026-02-03)

### Основные изменения

✅ **Убран пароль** - теперь хранилище работает без шифрования (простой JSON)
✅ **Добавлен поиск eSIM по номеру телефона** - команда `profile find-by-phone <MSISDN>`
✅ **Пользователь вводит SM-DP+ адрес и код активации** - команда `profile add` теперь требует `--smdp` и `--activation-code`
✅ **Защита от повторного использования activation code** - коды отслеживаются и блокируются после первого использования
✅ **Улучшена обработка ошибок** - детальные сообщения об ошибках с предупреждениями
✅ **Оптимизирован код** - профессиональная структура, типизация, документация
✅ **SMS использует MSISDN из активного профиля** - более реалистичная эмуляция

### Критические улучшения безопасности

- ⚠️ Activation codes теперь отслеживаются как одноразовые
- ⚠️ Если код уже использован, выводится предупреждение
- ⚠️ Если скачивание профиля завершится ошибкой, код становится недействительным

### Новые команды

- `esim profile find-by-phone <MSISDN>` - поиск профиля по номеру телефона
- `esim profile add --smdp <ADDRESS> --activation-code <CODE> [--msisdn <PHONE>]` - обновленная команда добавления профиля

### Удалено

- ❌ Пароль для хранилища (--password, ESIM_PASSWORD)
- ❌ Шифрование данных (теперь простой JSON)
- ❌ Зависимость cryptography

### Технические детали

- Все данные хранятся в `~/.esim_cli/state.json`
- Используется только стандартная библиотека Python
- Код оптимизирован и протестирован
